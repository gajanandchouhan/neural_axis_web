# Neural Axis Technologies — static website

This is a minimal static website scaffold for *Neural Axis Technologies* (Slogan: "The Core of Intelligent Innovation"). It focuses on mobile app development services.

Files created:
- `index.html` — homepage
- `css/style.css` — styles
- `assets/README.txt` — instructions for logo and additional assets

To preview locally:

```bash
# from this directory
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```

Add the provided logo image to `assets/logo.png` (PNG recommended). The page contains an inline SVG fallback if the logo isn't present.
